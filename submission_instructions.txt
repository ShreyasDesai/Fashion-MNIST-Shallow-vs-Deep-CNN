========================================================================
    HOW TO PUBLISH AND SUBMIT YOUR GRADED CNN ASSIGNMENT
========================================================================

Follow these simple steps to push your completed assignment to GitHub and share the link on Vlearn.

STEP 1: INITIALIZE GIT AND PUSH TO GITHUB
------------------------------------------
1. Open Git Bash, Command Prompt, or PowerShell.
2. Navigate to your project folder:
   cd "C:\Shreyas\antigravity\HeroVired\Graded Assignment on Convolutional Neural Networks (CNN)"

3. Initialize a new Git repository:
   git init

4. Add all files to staging:
   git add .

5. Commit the files locally:
   git commit -m "Complete Fashion-MNIST Shallow vs Deep CNN Study"

6. Go to https://github.com/ and sign in.
   - Click the "New" button to create a new repository.
   - Name your repository: "Fashion-MNIST-Shallow-vs-Deep-CNN"
   - Leave it PUBLIC. Do NOT check "Add a README", "Add .gitignore", or "Choose a license" (since we already have our files).
   - Click "Create repository".

7. Copy the repository link (e.g., https://github.com/YOUR_USERNAME/Fashion-MNIST-Shallow-vs-Deep-CNN.git).

8. Link your local directory to GitHub and push your files (replace <YOUR_REPO_URL> with the link you copied):
   git branch -M main
   git remote add origin <YOUR_REPO_URL>
   git push -u origin main
